# Application MERN de réservation de terrains sportifs

Application web développée avec le stack MERN (MongoDB, Express, React, Node.js).

## Fonctionnalités
- Réservation de terrains sportifs
- Gestion des utilisateurs
- Tableau de bord BI (Power BI)
- Analyse des KPI

## Technologies
- React
- Node.js / Express
- MongoDB
- Power BI
